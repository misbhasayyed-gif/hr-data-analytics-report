1. Introduction & Project Overview

1.1 Project Objectives

This dashboard presents key insights into employee salary distribution and attrition rates across different departments and locations. The goal is to provide a clear, interactive visualization that supports data-driven decision-making for HR management.

2. Data Description

2.1 Data Overview

The dataset comprises employee records from multiple departments and locations. It includes details such as salaries, attrition status, and work locations. This structure enables the analysis of salary distribution, attrition trends, and department/location-wise workforce patterns.

2.2 Data Fields

ID: Unique employee identifier.

Department: The functional area of the employee (HR, Sales, IT, Finance).

Salary: The employee's annual salary.

Attrition Status: Indicates if the employee left the company (Yes/No).

Location: The city of posting (Mumbai, Pune, Delhi).

2.3 Analysis Dimensions

The dataset allows analysis across different dimensions such as:

  • Department-wise salary and attrition patterns
  • Location-based workforce insights
  • Overall attrition trends across the organization

2.4 Data Quality and Sources

The dataset is well-structured, complete, and suitable for HR analytics. It provides reliable insights for salary comparison, attrition monitoring, and workforce planning at a macro level.

4. Methodology

1. Analytical Approach

The project uses a comparative analysis, examining average salaries and attrition rates across departments and locations to identify trends and key insights.

2. Data Collection & Validation

Employee data was sourced from internal HR records and validated for completeness and consistency across all fields (Salary, Department, Attrition Status, Location).

3. Trend Analysis

Pivot tables and charts were used to analyze salary distribution and attrition patterns, highlighting variations across departments and cities.

4. Visualization Creation

Charts were chosen based on the type of insight:

  • Bar Charts for department-wise salary comparison
  • Pie Charts for attrition proportion

5. Narrative Development

The dashboard is structured to guide users from general workforce patterns to specific insights, e.g., high-attrition departments or locations with salary disparities.

4.1 Tools and Technologies Used

  1. Microsoft Excel: For data organization, pivot table creation, and charting.
  2. Excel Pivot Tables: To summarize and analyze data dynamically.
  3. Excel Charts: For visual representation of data trends and comparisons.
  4. Data Analysis Techniques: Comparative metrics, averages, and percentage calculations.
  5. Dashboard Design Principles: User-friendly layout, clear labeling, and interactive elements (slicers).

5. Conclusion and Key Insights

The project demonstrates that salary and attrition trends vary significantly across departments and locations. By applying data storytelling, HR managers can quickly identify:

  • Departments with higher attrition rates that may need targeted retention strategies.
  • Locations where salary adjustments could improve employee satisfaction and reduce turnover.
  • Overall workforce patterns that inform strategic HR planning and decision-making.

6. Implications for the Project & Learning Outcomes

This project reinforces skills in the data lifecycle: sourcing, validating, analyzing, visualizing, and narrating HR data. The key learning outcome is the ability to create interactive dashboards that clearly communicate analytical findings to support HR decision-making.
